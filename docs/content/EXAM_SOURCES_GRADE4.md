# Nguồn tham khảo đề thi lớp 4 (HK1 / HK2) — 2020–2025

Tài liệu liệt kê **55 nguồn công khai** (bộ đề, đề trường, thư viện số) dùng làm **ma trận chủ đề** khi soạn ngân hàng câu hỏi game.  
**Không** sao chép nguyên văn đề có bản quyền; chỉ trích **dạng bài / chủ đề** phù hợp chương trình lớp 4 (Thông tư 22 / 27, SGK *Kết nối tri thức*, *Chân trời sáng tạo*).

Phạm vi năm: **2020-2021 → 2024-2025** (đủ 5 năm học gần đây).

## Cách dùng trong repo

| Hạng mục | Vị trí |
|----------|--------|
| Sinh lại câu hỏi từ chủ đề | `npm run generate:questions` → `scripts/generate-grade4-question-banks.mjs` |
| Ngân hàng đã gắn game | `*Supplement.ts`, `mcqBank.ts`, `mathBank.ts` trong `src/games/` |
| Chỉ mục máy đọc được | `scripts/exam-sources-grade4.json` |

## Cập nhật ngân hàng

Lần mở rộng wave 2 (sau 55 nguồn): tăng mục tiêu sinh lên **~200 câu / ngân hàng chính** qua `*Extra.ts` trong script. Chạy:

```bash
npm run generate:questions
npm run build
```

## Lưu ý bản quyền

- Chỉ dùng nguồn để nắm **cấu trúc đề** và **chủ đề chương trình**.
- Câu hỏi trong game là nội dung **soạn mới**, không trích nguyên đề trường/tổ chức thương mại.

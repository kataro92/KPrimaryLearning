# Bút Sen Việt — hero glTF

- **Model:** [Low poly pen](https://sketchfab.com/3d-models/low-poly-pen-e63063d93b894b42952277574bb7860d)
- **Author:** [VertexKiller (@alexdemash94)](https://sketchfab.com/alexdemash94)
- **License:** [CC BY 4.0](http://creativecommons.org/licenses/by/4.0/)

Place **`pen.glb`** or a glTF package with entry **`scene.gltf`** in this folder.

Download (API token required):

```bash
SKETCHFAB_API_TOKEN=… npm run fetch:models
```

Without the file, the game uses a procedural fountain-pen + lotus accent in `butSenPenScene.ts`.

# Hội An — glTF assets

Models live in subfolders so lantern and boat do not overwrite each other.

## `lantern/` — paper lantern (treo khi trả lời đúng)

- **Model:** [Japanese paper lantern](https://sketchfab.com/3d-models/japanese-paper-lantern-4450069fee444ae0920aa5babd5d9239)
- **Author:** [@abelGB](https://sketchfab.com/abelGB)
- **License:** [CC BY 4.0](http://creativecommons.org/licenses/by/4.0/)

Place **`scene.gltf`** (+ `scene.bin`, `textures/`) or **`paper-lantern.glb`** in `lantern/`.

## `boat/` — fishing boat (hero thuyền)

- **Model:** [Fishing Boat (asset)](https://sketchfab.com/3d-models/fishing-boat-asset-8a22e9c369d64a329c67e9a355064b2c)
- **Author:** [@LucSmits](https://sketchfab.com/LucSmits)
- **License:** CC BY 4.0

Place **`scene.gltf`** (+ `scene.bin`) or **`fishing-boat.glb`** in `boat/`.

Download via `npm run fetch:models` (see root `README.md`).

Without files, the game uses procedural đèn lồng / thuyền.

# Bảng Cửu Chương Văn Miếu — glTF

- **Model:** [Sea turtle low poly](https://sketchfab.com/3d-models/sea-turtle-low-poly-c92be549c8194136914883309a13a6b5)
- **Author:** [@C.J..Goldman](https://sketchfab.com/C.J..Goldman)
- **License:** [CC BY 4.0](http://creativecommons.org/licenses/by/4.0/)

Place **`scene.gltf`** (+ `scene.bin`, `textures/`) or **`turtle.glb`** here.

Download: `SKETCHFAB_API_TOKEN=… npm run fetch:models`

Without the file, the hero pane uses a procedural green turtle.

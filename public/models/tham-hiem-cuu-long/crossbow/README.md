# Nỏ Minecraft — glTF asset (Nhà thám hiểm Cửu Long)

Place the downloaded model here:

- **`crossbow.glb`** — single-file export (optional)
- **`scene.gltf`** (+ `scene.bin`, `textures/`) — default from `npm run fetch:models`

## Source (Sketchfab)

- **Model:** [Crossbow (Minecraft)](https://sketchfab.com/3d-models/crossbow-minecraft-c071665dc11846ffb69665411cbad428)

## Download

```bash
SKETCHFAB_API_TOKEN=your_token npm run fetch:models
```

If the file is missing, the game uses a lightweight procedural voxel crossbow in the FPS view.
